# 🏠 Zot Dorms

A modern full-stack web application that allows students to explore, review, and compare UCI dorms. Users can browse dorm listings, view ratings, and navigate to detailed review pages for each dorm.

---

## 🚀 Features

- 🏫 Browse dorm listings with images, pricing, and ratings
- ⭐ View dorm ratings and descriptions
- 🔍 Responsive grid layout for clean UI across devices
- 🔗 Dynamic routing for individual dorm pages
- 🎨 Modern UI inspired by real-world platforms

---

## 🛠️ Tech Stack

**Frontend**
- React (with TypeScript)
- Vite
- CSS (custom styling)
- React Router

**Tools & Libraries**
- react-icons
- modern CSS (Grid, Flexbox)
